# Data-Science-Alura-Challenge

## Desafio de Análise de Dados – Lojas Alura Store

### Sobre o Projeto

Este projeto foi desenvolvido como parte de um desafio da **Alura** no curso de **Python para Data Science**.
O objetivo principal é aplicar técnicas de análise exploratória de dados (EDA) utilizando bibliotecas como **Pandas** e **Matplotlib**, explorando o desempenho de quatro empresas fictícias em diferentes categorias de vendas.

Ao longo do projeto, são realizadas análises quantitativas e visuais para compreender os principais indicadores de desempenho das lojas, tais como faturamento, avaliações de clientes, frete médio e produtos mais vendidos.

### Tecnologias Utilizadas

- Python

- Pandas – manipulação e análise de dados

- Matplotlib – criação de gráficos e visualizações

Jupyter Notebook – ambiente interativo para execução e visualização das análises

## Principais Análises

O projeto foi dividido em etapas que cobrem diferentes aspectos do desempenho das lojas.

### 1. Faturamento Total das Lojas

Soma do valor total de vendas realizadas por cada loja.

Permite identificar qual loja teve o melhor desempenho financeiro.

### 2. Categorias de Produtos Mais e Menos Vendidas

Análise da distribuição das vendas por categoria de produto.

Identificação dos produtos com maior e menor volume de vendas.

### 3. Média das Avaliações de Clientes por Loja

Cálculo da média das notas de avaliação deixadas pelos clientes.

Indica o nível de satisfação geral com os produtos e serviços.

### 4. Produtos Mais e Menos Vendidos

Determinação dos produtos com maior e menor número de vendas em cada loja.

Ajuda a entender a preferência dos clientes e o desempenho do portfólio.

### 5. Frete Médio de Cada Loja

Cálculo do custo médio de frete por venda.

Pode indicar diferenças logísticas ou de estratégia comercial entre as lojas.
